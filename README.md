# BlinkenClock
A Wall-Clock based on an Arduino and RGB-LED Stripe.


## Features
* RGB-LED Stripe with 60 LEDs/Pixels
* LDR Light Sensor
* DS1307 RTC (Real-Time-Clock) with backup battery
